public subroutine kepler_shepperd(mu, rv1, dt, rv2, istat)

Kepler propagation using Shepperd's method.

Reference

  • S.W. Shepperd, "Universal Keplerian State Transition Matrix". Celestial Mechanics 35(1985) p. 129-144.
  • Rody P.S. Oldenhuis, progress_orbitM Matlab function (BSD license). http://www.mathworks.com/matlabcentral/fileexchange/26349-kepler-state-transition-matrix-mex
  • C.D. Eagle, Orbital Mechanics with MATLAB, twobody2.m Matlab function (BSD license). http://www.mathworks.com/matlabcentral/fileexchange/48723-matlab-functions-for-two-body-orbit-propagation
